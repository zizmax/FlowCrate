Spotify

Required for matching, playback, queue, and playlist actions.

First time? Create your Spotify app (2 minutes)
  1. Go to developer.spotify.com/dashboard and log in with the Spotify account you actually listen with (Premium is required for playback).
  2. Click Create app. Name and description can be anything (e.g. "Flow Crate"). Set the Redirect URI to exactly {{ cfg.spotify_redirect_uri }}, and check the Web API box.
  3. Open the app's Settings, then copy the Client ID and Client Secret into the fields below. Click Save Settings, then Test Spotify.

Flow State

Public Flow State posts work by default. For paid posts, Flow Crate automatically reuses your logged-in browser session when a post is paywalled. (Your computer may ask once for access to read those cookies.) If the automatic session detection fails, you need to paste your Substack SID cookie here as a manual fallback.

How to find your Substack SID
  1. In the browser where you're logged in to your Flow State subscription, open flowstate.fm.
  2. Open developer tools (Cmd+Option+I in Chrome/Firefox; in Safari enable the Develop menu first) and go to the storage/cookies view (Application → Cookies in Chrome).
  3. Select the flowstate.fm site, find the cookie named substack.sid, and copy its value into the Substack SID field above, then Save Settings.

Sonos & Siri (optional)

Play the latest post directly on a Sonos speaker, including from an Apple Shortcut ("Hey Siri, play Flow Crate"). Set an API token to enable the /api/play-latest endpoint.

Download Siri Shortcut

Open this Settings page in Safari on your iPhone (http://{{ local_hostname }}:{{ local_port }}/settings), tap Download Siri Shortcut, open the downloaded file, then tap Add Shortcut. Then say: "Hey Siri, Play Flow Crate".

Set up the Shortcut manually

The phone must be on the same Wi-Fi network as this computer.

On your iPhone, open Shortcuts, create a new shortcut named exactly Play Flow Crate (the name is the Siri phrase), and add these four actions:

  1. Get Contents of URL
    • URL: http://{{ local_hostname }}:{{ local_port }}/api/play-latest
    • Method: POST
    • Header: X-FlowCrate-Token = your API token (above)
    • No request body is needed — the endpoint accepts an empty body and plays immediately.
  2. Get Dictionary Value — key speak from the previous action's output.
  3. Speak Text — the Dictionary Value (always read aloud).
  4. Show Result — the Dictionary Value (shown in the Siri card).

Then say "Hey Siri, Play Flow Crate." Music starts on the configured Sonos room and Siri reads the post title, date, and track count.
